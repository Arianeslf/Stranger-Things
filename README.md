# Stranger Things Experience

Projeto desenvolvido inspirado no universo de **Stranger Things**, com uma interface visual temática, animações e seções interativas para apresentar uma experiência imersiva ao usuário.

🔗 Acesse o projeto: https://stranger-things-zeta.vercel.app/

## Sobre o projeto

O site apresenta uma página temática da experiência Stranger Things, com destaque para escolha de cidades, cards informativos, depoimentos e uma identidade visual inspirada na série.

O objetivo foi praticar estruturação de páginas com HTML, CSS e JavaScript, além de utilizar animações com GSAP para deixar a navegação mais dinâmica.
